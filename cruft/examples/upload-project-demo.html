Upload Project Demo

Examples of creating/opening projects using the Workspace API

Step 1: Check Workspace

Get Workspace Information

First, perform a handshake to check the workspace status and capabilities.

Click the button to check workspace...

Step 2: Upload a Project (ZIP)

Upload ZIP File to Workspace

Upload a ZIP file containing your project. It will be extracted to the workspace root.

Note: The ZIP file should contain your project files. The contents will be extracted to the workspace directory specified on the server.
Select a ZIP file to upload...

Step 3: Browse Uploaded Files

List Files in Workspace

Browse the files in your workspace after uploading.

Click the button to browse files...

Step 4: Start Debug Session

Start Debugging a File

Start a debug session for a JavaScript file in your workspace.

Click the button to start debugging...

Code Examples

Example 1: Check Workspace Info

const response = await fetch('http://localhost:8080/workspace/info'); const workspaceInfo = await response.json(); console.log('Workspace Root:', workspaceInfo.workspaceRoot); console.log('Features:', workspaceInfo.features);

Example 2: Upload ZIP Project

const formData = new FormData(); formData.append('file', zipFile); const response = await fetch('http://localhost:8080/project/', { method: 'POST', headers: { 'X-Workspace-API-Key': 'dev-key-123', 'Content-Type': 'application/zip' }, body: zipFile // Send ZIP file directly }); const result = await response.json(); console.log('Uploaded:', result.filesExtracted, 'files');

Example 3: Start Debug Session

const response = await fetch('http://localhost:8080/debug/session', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ file: 'test/fixtures/steppable-script.js' }) }); const session = await response.json(); console.log('Session ID:', session.session.sessionId); console.log('WebSocket URL:', session.session.wsUrl);
